# 🤖 ChatGPT 本地调用脚本

本脚本允许你在本地通过命令行与ChatGPT进行交互，无需打开浏览器即可使用强大的AI助手。

[English Documentation](en/README.md)

## 🚀 功能特点

- 🔑 支持API密钥配置（环境变量或直接传参）
- 💬 实时流式输出响应
- 📝 保存和加载对话历史
- 🛠️ 可自定义系统提示词
- 🔄 支持多种模型选择

## 📋 使用前提

1. 拥有OpenAI API密钥 (https://platform.openai.com/api-keys)
2. 安装Python 3.6+

## 📦 安装依赖

```bash
1 # 创建并激活虚拟环境
2 python3 -m venv chatgpt_env
3 source chatgpt_env/bin/activate
4 # 安装依赖包
5 pip install openai python-dotenv
```

## 🔧 配置

有两种方式配置API密钥:

1. 创建`.env`文件，添加以下内容:

```
1 OPENAI_API_KEY=你的OpenAI密钥
```

2. 通过命令行参数传入:

```bash
1 python chatgpt.py --key 你的OpenAI密钥
```

## 📝 使用方法

### 基本使用

```bash
1 # 确保在LLM目录中
2 cd LLM
3 # 激活虚拟环境
4 source chatgpt_env/bin/activate
5 # 运行脚本
6 python chatgpt.py
```

### 高级选项

```bash
1 python chatgpt.py --model gpt-4 --system "你是一个有用的AI助手" --save conversation.json
```

参数说明:
- `--key`: OpenAI API密钥 (如不提供则使用环境变量)
- `--model`: 模型名称 (默认: gpt-3.5-turbo)
- `--system`: 系统提示词
- `--load`: 加载对话历史文件
- `--save`: 保存对话到指定文件

## 💡 对话中的命令

- 输入`exit`退出程序
- 输入`save`手动保存当前对话
